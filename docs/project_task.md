一、竞赛背景与主题
某制造企业主要从事精密行星减速器的研发设计与生产制造，随着机器人与自动
化设备等的应用需求，精密行星减速器向小型化、轻量化、提高传动效率、降低噪音
等方向发展。在精密行星减速器的制造环节，各核心零件涉及毛坯制造（铸造或锻造
）、机械加工（铣削、钻孔等）、轴类零件加工（切割、车削等）、尺寸检测（三坐
标）等多工艺多工序，该企业生产车间涉及车铣磨、CNC、热处理、三坐标等多品类、
多品牌的加工和检测设备。
随着人形机器人等行业的快速发展，该企业的订单业务将呈现良好发展趋势，如
何利用数字化系统来高效地管理生产设备，并提升工序、工艺的设计和执行效率，成
为该企业须重点解决的问题之一。（附：快速了解工艺管理系统中设备、物料、工序、

1、配置工序
任务描述：根据行星减速器核心零件生产工艺流程图（毛坯制造→机械加工<粗加
工>→轴类零件加工<精加工>→检测→入库）来配置5道工序，工序名称分别为：毛坯
制造、粗加工、精加工、检测、入库。
任务成果：提交工程代码zip文件、调用xDM-F的接口列表和功能演示录屏。
2、配置工艺路线
任务描述：根据工艺流程（毛坯制造→机械加工→轴类零件加工→尺寸检测→入
库），新增一条工艺路线，名称为“中心轮零件加工”，版本 1.0，主要涉及与行星
减速器相关零件的生产加工工艺。 工艺安排顺序包含多个工序环节，涵盖从初步处理
到最终步骤的各类操作，流程涉及一些如加工、检测与入库等步骤。
任务成果：提交工程代码zip文件、调用xDM-F的接口列表和功能演示录屏。
3、工业数据建模要求
华为工业数字模型驱动引擎iDME-数据建模引擎（xDM Foundation，简称xDM-F），
提供“设计即开发、功能可配置、模型驱动”的一体化数据模型管理能力，支撑数据
模型全生命周期操作。
本次大赛要求在xDM-F进行所有数据模型的设计和开发。
任务描述：
（1）设计并开发设备模型，推荐中英文名称为：设备/Equipment，属性包含设备
编码、设备名称、生产厂家、品牌、规格型号、供应商、生产日期、使用年限、折旧
方式、位置等，并支持按需自定义技术参数信息、备品备件信息等扩展属性。
（2）设计并开发物料模型，推荐中英文名称为：部件/Part，属性包含物料编号、
物料名称、规格型号、库存数量、供应商等，并支持分类管理、版本管理。
（3）设计并开发工序模型，推荐中英文名称为：工序/WorkingProcedure，属性
包含工序编号、工序名称、生产步骤、生产和检测设备、操作人员、开始时间、结束
时间等。
